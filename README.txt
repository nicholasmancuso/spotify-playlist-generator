This project builds a web application that connects to the Spotify API to generate playlists based on mood, genre, or artist input. The app uses the Spotify PKCE authentication flow for secure login. Users select a mode, enter details, and receive a custom playlist. The outcome is an interactive tool that simplifies music discovery and personalization.
